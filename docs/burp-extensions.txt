# Burp Suite BApp Extensions for 2FA Testing Workflow
# Recommended extensions untuk complement TfaBuster

## ESSENTIAL (Must Have) ⭐⭐⭐⭐⭐

### 1. Turbo Intruder
**Rating**: ⭐⭐⭐⭐⭐
**Category**: Professional Community
**Use Case**: Race condition testing - gate mechanism untuk simultaneous requests
**Why**: Alternative manual untuk TfaBuster's race_async.py module
**Install**: Burp → Extender → BApp Store → Search "Turbo Intruder"
**Link**: Send large numbers of HTTP requests and analyze the results

### 2. Copy as Python httpx
**Rating**: ⭐⭐⭐⭐⭐
**Category**: Professional Community
**Use Case**: Export requests as Python httpx code (EXACT sama dengan TfaBuster!)
**Why**: Instant payload generation - langsung compatible dengan race_async.py
**Install**: Burp → BApp Store → "Copy as Python httpx"
**Link**: Export HTTP/1.1 and HTTP/2 requests for use in Python

### 3. Auth Analyzer
**Rating**: ⭐⭐⭐⭐⭐
**Category**: Professional Community
**Use Case**: Repeat requests dengan different headers/tokens, detect auth bypass
**Why**: Perfect untuk pre-scan sebelum TfaBuster automated testing
**Install**: Burp → BApp Store → "Auth Analyzer"
**Link**: Find authorization bugs by repeating Proxy requests with self defined headers

## WORKFLOW INTEGRATION

**Step 1: Manual Discovery (Burp + Extensions)**
```
1. Use Auth Analyzer → identify weak auth
2. Use Additional CSRF Checks → flag CSRF issues
3. Use JWT Editor → test token manipulation
4. Use Turbo Intruder → manual race condition
```

**Step 2: Export to TfaBuster**
```
Right-click request → Extensions → Copy as Python httpx
Paste directly into TfaBuster test case or as --payload
```

**Step 3: Automated Scan (TfaBuster)**
```bash
python main.py --target <url> --payload <from-burp>
```

**Step 4: Compare & Report**
```
Manual findings (Burp) + Automated (TfaBuster) = Complete assessment
```
